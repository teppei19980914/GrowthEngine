# ユメログ 仕様書

## 1. テーマとコンセプト

### テーマ

**「夢を持つことで、大きな壁も乗り越えられる」**

### コンセプト

ユメログは **「止まっている人を動かし、動いている人を支える」** アプリです。

人は慣性で行動します。動いている人は動き続け、止まっている人は止まり続けます。ユメログは「最初の一歩」を踏み出すきっかけと、動き続けるための仕組みを提供します。

#### 開発の原点

開発者は学生時代に夢を持てず、将来に希望を見出せない経験をしました。本との出会いを通じて「生き生きと生きるには夢が必要」と気づき、夢を言語化し行動に移す仕組みをアプリとして形にしました。

#### 3つの柱

1. **言語化**: 夢を言葉にして書き出すことで、脳がそれを「目標」として認識する
2. **分解**: 夢を目標→タスクに分解し、日々の行動レベルまで具体化する
3. **継続**: 活動の積み重ねを星座として可視化し、モチベーションを維持する

#### 設計思想

- **高機能をシンプルに**: エンジニア向けの専門用語を避け、誰でも直感的に使えるUI
- **ハードコーディング排除**: 全テキストを外部ファイルで管理し、保守性を最大化
- **インフラ非依存**: 開発者通知やリリース情報はアプリ内ファイルで管理し、特定インフラに依存しない

## 2. 画面仕様

### 2.1 ダッシュボード（ホーム）

| 項目 | 仕様 |
|---|---|
| パス | `/` |
| 機能 | カスタマイズ可能なウィジェットグリッド |
| ウィジェット数 | 12種類（受信ボックス、今日の活動、合計時間、活動日数、夢の数、目標数、連続活動、本棚、自己ベスト、実施率、アクティビティ、星座） |
| 編集モード | ドラッグハンドルで並べ替え、サイズ変更（1列/2列）、追加/削除 |
| 時間帯挨拶 | 0-5時: お疲れさまです、5-12時: おはようございます、12-18時: こんにちは、18時以降: こんばんは |

### 2.2 夢ページ

| 項目 | 仕様 |
|---|---|
| パス | `/dreams` |
| 登録項目 | タイトル（必須）、説明、Why（動機）、カテゴリ（8種） |
| カテゴリ | キャリア、学び、健康、お金、趣味、人間関係、旅行、その他 |
| 発見ガイド | 質問回答形式でカテゴリを推定し、テンプレートを提案 |

### 2.3 目標ページ

| 項目 | 仕様 |
|---|---|
| パス | `/goals` |
| 登録項目 | What（必須）、When（日付/期間指定）、How、紐づく夢（任意）、カラー |
| 並べ替え | ドラッグ&ドロップ対応 |
| 進捗表示 | 円形プログレスリング（タスク完了率） |
| 期限表示 | 超過: 赤、30日以内: オレンジ |

### 2.4 スケジュールページ

| 項目 | 仕様 |
|---|---|
| パス | `/gantt` |
| 表示モード | 全タスク、目標別、全書籍 |
| 日付範囲 | 直近3ヶ月（デフォルト）、6ヶ月、1年、全期間 |
| スクロール | マウスホイール→横、Shift+ホイール→縦、タッチドラッグ→両方向 |
| 初期表示 | 今日の位置にスクロール |
| 書き出し | HTML、Excel (.xlsx)、CSV |
| 日付ジャンプ | 「今日」ボタン、日付選択ピッカー |
| タスクタップ | 選択メニュー（編集/活動記録） |
| 左列表示 | グループ先頭行: 目標名（上段）+タスク名（下段）、2行目以降: タスク名のみ |

### 2.5 書籍ページ

| 項目 | 仕様 |
|---|---|
| パス | `/books` |
| 表示形式 | 本棚風UI（背表紙が見える書籍カバー） |
| 書籍幅 | 画面幅に応じて動的計算（40-60px） |
| ステータス | 未読、読書中、読了 |
| カテゴリ | IT・技術、ビジネス、小説・文学、自己啓発、学術・教育、趣味・実用、その他 |
| 読了レビュー | 読了日、要約、感想 |

### 2.6 星座ページ

| 項目 | 仕様 |
|---|---|
| パス | `/constellations` |
| 星座数 | 12（黄道十二星座） |
| 進捗基準 | 活動時間 ÷ 星あたり時間 = 灯る星の数 |
| 完成表示 | 金色バッジ + 説明テキスト |

### 2.7 統計ページ

| 項目 | 仕様 |
|---|---|
| パス | `/stats` |
| セクション | サマリー、自己ベスト、実施率、目標別統計（プレミアム）、書籍統計、アクティビティチャート（プレミアム）、最近の活動ログ |
| 期間切替 | 日、週、月、年 |

### 2.8 設定ページ

| 項目 | 仕様 |
|---|---|
| パス | `/settings` |
| セクション | 外観、通知設定、データ管理、アカウント、アプリ情報 |
| アプリ情報 | アプリ名、ご利用プラン、バージョン、最終更新日、プラットフォーム |

## 3. 受信ボックス仕様

### 通知種別

| 種別 | アイコン | データソース | 動作 |
|---|---|---|---|
| リマインド | アラーム（赤） | ユーザーのタスク/目標期限 | アプリ起動時に自動チェック |
| 実績 | トロフィー（黄） | ユーザーの活動記録 | マイルストーン達成時に自動生成 |
| お知らせ | メガホン（青） | `announcements.json` | アプリ起動時にDBと完全同期 |

### リマインダー検知ルール

| 条件 | 通知タイトル |
|---|---|
| 期限7日以内 | 「○○の期限まであとN日」 |
| 期限当日 | 「○○の期限は今日です」 |
| 期限超過 | 「○○が期限を過ぎています」 |

### タップ時の動作

| 通知種別 | 動作 |
|---|---|
| リマインド（タスク） | タスク編集ダイアログを直接表示 |
| リマインド（目標） | 目標ページに遷移 |
| 実績/お知らせ | 詳細ダイアログを表示 |

## 4. 体験版制限仕様

### レベル別制限値

| レベル | 条件 | 夢 | 目標（全体） | タスク（各目標） | 書籍 |
|---|---|---|---|---|---|
| 0 | 初期状態 | 1 | 3 | 3 | 3 |
| 1 | フィードバック1回 | 2 | 5 | 5 | 4 |
| 2 | フィードバック2回 | 3 | 8 | 8 | 5 |
| 3+ | フィードバック3回以上 | 無制限 | 無制限 | 無制限 | 無制限 |

### プレミアム解放条件

- サブスクリプション契約
- 招待コード入力
- 開発者メールアドレスでの認証

## 5. データ管理仕様

### 管理ファイル

| ファイル | 役割 |
|---|---|
| `pubspec.yaml` | ビルドバージョン |
| `lib/app_version.dart` | 表示バージョン + リリース履歴 |
| `lib/l10n/app_labels.dart` | 全UIテキスト |
| `assets/announcements.json` | 開発者通知（受信ボックス同期） |

### クラウド同期

- Firebase Auth（匿名認証 + メール連携）
- Firestore（全データJSON形式で同期）
- タイムスタンプ比較でダウンロード/アップロードを自動判定
